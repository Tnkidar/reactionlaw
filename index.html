反應速率的基礎概念

本區塊介紹化學反應速率的核心定義與數學表示法。透過下方的動態折線圖，您可以直觀地觀察反應物濃度如何隨時間減少，以及生成物濃度如何隨時間增加。了解這些基礎，有助於掌握後續更複雜的動力學概念。

核心定義

單位時間內，反應物濃度的消耗量或生成物濃度的生成量。

反應物：$r = - \frac{\Delta[\text{反應物}]}{\Delta t}$
生成物：$r = \frac{\Delta[\text{生成物}]}{\Delta t}$

單位：$M \cdot s^{-1}$ 或 $atm \cdot s^{-1}$

濃度對時間變化關係圖

範例反應：$2NO_2(g) \rightarrow 2NO(g) + O_2(g)$

反應速率與係數關係

此區塊探討同一個化學反應中，不同物質的反應速率之間的數學關聯。您可以利用下方的互動計算機，輸入自訂的化學計量係數與某一物質的速率，系統將自動推算出其他物質的速率，協助您驗證「速率比等於係數比」的法則。

理論關係

對於一般化學反應：
$aA + bB \rightarrow cC$

  • 速率比 = 係數比
    $r_A : r_B : r_C = a : b : c$
  • 關係等式
    $\frac{1}{a} r_A = \frac{1}{b} r_B = \frac{1}{c} r_C$
  • 氣體總壓
    若反應前後氣體分子數不等（$\Delta n \ne 0$），可用總壓變化速率來表示整體反應速率。

互動速率計算機

A + B $\rightarrow$ C

$r_C = $ M/s

反應速率的測量方法

不同的化學反應具有不同的巨觀特性。本區塊整理了實驗室中常見的五種測量反應速率的物理指標。點擊各個卡片，即可了解該測量方法的適用條件與常見的化學反應實例。

🎨

顏色深淺

適用於反應物或生成物具有明顯顏色的系統。

例: $N_2O_4 \rightarrow 2NO_2$
(無色 $\rightarrow$ 紅棕色)
🎈

總壓 / 總體積

適用於反應前後氣體分子數「不相等」的系統 ($\Delta n \ne 0$)。

例: $2N_2O_5 \rightarrow 4NO_2 + O_2$
(氣體總數增加，總壓上升)
⚡

導電度

適用於水溶液中離子數目或種類有顯著變化的系統。

例: 反應產生沉澱或將強電解質變為弱電解質，導致導電度下降。
🧪

pH 值

適用於反應過程中涉及 $H^+$ 或 $OH^-$ 消耗或生成的系統。

例: $C_2H_5OH + HI \rightarrow C_2H_5I + H_2O$
(酸性減弱，pH上升)
⬇️

沉澱量

適用於產生固體沉澱的反應，藉由測量沉澱物質量變化推算。

例: $Na_2S_2O_3 + 2HCl \rightarrow 2NaCl + SO_2 + S\downarrow + H_2O$
(測量硫黃色沉澱)

速率定律與速率常數

本區塊解析反應速率與反應物濃度之間的數學關係，即「速率定律式」。特別注意，反應級數無法單從反應方程式看出，必須由實驗數據推導。透過下方的表格，您可以清楚歸納速率常數 $k$ 的單位如何隨著反應總級數而改變。

1. 速率定律式

$r = k[A]^m[B]^n$
  • $m, n$ 反應級數：必須由實驗數據求得，不一定等於平衡係數。
  • 總級數： $m + n$
  • 平均速率 vs 瞬時速率：
    平均：濃度對時間圖的「割線斜率」。
    瞬時：濃度對時間圖的「切線斜率」。

2. 速率常數 ($k$)

影響因素：反應物本性、溫度、催化劑。

※ 濃度、壓力、反應熱皆不影響 $k$ 值。

$k$ 的單位通式：$M^{1-n} \cdot t^{-1}$

反應總級數 ($n$) 單位 (以 M, s 為例)
0 級 $M \cdot s^{-1}$
1 級 $s^{-1}$
2 級 $M^{-1} \cdot s^{-1}$
3 級 $M^{-2} \cdot s^{-1}$
